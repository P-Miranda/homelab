# Gotify
Simple server to send and receive notifications

- [Official documentation](https://gotify.net/docs/)

## Setup
```bash
cp .env.example .env
# edit .env with correct variable values
# run docker
docker compose up -d
```
- login to WebUI:
    - [VM IP]:PORT (default 8080)
    - use password in `.env`
- create users, applications, etc
